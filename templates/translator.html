001 : Translator

Translator which translates English words and files to another languages.

"Without translation, we would be living in provinces bordering on silence."

George Steiner

With the help of this translator you can translate words, sentences and even entire files from English to any others. There is a example of translating .txt file from English to Russian.

# Translator

from translate import Translator
translator = Translator(to_lang="ru")
try:
    with open('./test.txt', mode="r") as my_file:
        text = my_file.read()
        translation = translator.translate(text)
        with open('./test2.txt', mode="w") as my_file2:
            my_file2.write(translation)
        print(translation)
except FileNotFoundError as err:
  print("check your file path!")
